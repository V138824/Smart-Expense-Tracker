# 💰 Smart-Expense-Tracker
A Java Desktop Application for managing and tracking daily expenses. Developed as a Second Year Engineering Project.

## ✨ Features
- **Add Daily Expenses:** Input spending with categories.
- **View Transactions:** See a history of all expenses.
- **Total Expenditure:** Real-time calculation of total spent.
- **Category-wise Breakdown:** Analysis of spending patterns (Food, Travel, etc.).

## 🛠️ Tech Stack
- **Language:** Java
- **Framework:** Swing, AWT (GUI)
- **Database:** SQLite
- **Library:** sqlite-jdbc-3.51.3.0.jar

## 🚀 How to Run
1. Download all files or clone the repo.
2. Make sure the `sqlite-jdbc` JAR file is in your project classpath.
3. Run `ExpenseManager.java`.
4. To view the raw database, use [DB Browser for SQLite](https://sqlitebrowser.org/).
